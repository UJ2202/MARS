### Hover Effects
- **Cards**: Border glow + background lightening
- **Buttons**: Scale up (1.02) + shadow increase
- **Events**: Background color change
- **Files**: Show action buttons

- **Selected Tab**: Border-top (blue) + darker background
- **Expanded Event**: Visible details panel

- **Fetching Data**: Spinning loader (blue)
